# Double-Check Before Styling

Use this prompt when you are about to change page styling.

## Prompt
Before changing any styling, do a verification pass first.

1. Restate the user request in one short paragraph.
2. List the styling goals you believe are required.
3. Cross-check your plan against the request and note any missing requirements.
4. If anything is unclear, conflicting, or assumed, ask a clarification question before editing.
5. Proceed with styling changes only after the checks above are complete.

## Required Behavior
- Do not edit CSS, HTML classes, theme variables, layout spacing, or visual assets until the verification pass is done.
- If confidence is below high, pause and ask for clarification.
- Keep the final styling edits minimal and aligned to the verified requirements.
